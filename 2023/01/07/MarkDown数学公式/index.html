三、上下标、根号、分数、括号、上下取整

!!首先要知道由{}包围的内容只一个小单元!!，可以用来消除二义性

本节都只对前后一个单元生效

1、上下标

  • 上标：^
  • 下标：_

举例：

A^B ⇒\Rightarrow⇒ ABA^BAB

${AB}^{BC}$⇒\Rightarrow⇒ABBC{AB}^{BC}ABBC

${AB}^BC$⇒\Rightarrow⇒ABBC{AB}^BCABBC

A_B ⇒\Rightarrow⇒ ABA_BAB​

${AB}_{BC}$⇒\Rightarrow⇒ABBC{AB}_{BC}ABBC​

${AB}_BC$⇒\Rightarrow⇒ABBC{AB}_BCABB​C

  • 正上/下方：\limits_{xxx}^{xx}

举例：

$\iiint\limits_{xxx}^{xx}$ ⇒∭xxxxx\Rightarrow\iiint\limits_{xxx}^{xx}⇒xxx∭xx​

2、根号

$\sqrt[a]{b}$ ⇒\Rightarrow⇒ba\sqrt[a]{b}ab​

举例：

$\sqrt[a]b$ ⇒\Rightarrow⇒ba\sqrt[a]bab​$

$\sqrt[a]bc$ ⇒\Rightarrow⇒bac\sqrt[a]bcab​c

$\sqrt[a]{bc}$ ⇒\Rightarrow⇒bca\sqrt[a]{bc}abc​

3、分数

1、$\frac{a}{b}$⇒\Rightarrow⇒ab\frac {a}{b}ba​ 推荐

2、$a \over b$⇒\Rightarrow⇒aba \over bba​

举例：

$a+b \over b$⇒\Rightarrow⇒a+bba+b \over bba+b​

$a+{c} \over b$⇒\Rightarrow⇒a+cba+{c} \over bba+c​

4、括号

$()$⇒\Rightarrow⇒()()()

$[]$⇒\Rightarrow⇒[][][]

$\{\}$⇒\Rightarrow⇒{}\{\}{} 需要转义

$\langle \rangle$⇒\Rightarrow⇒ ⟨⟩\langle \rangle⟨⟩

5、上下取整

向上取整： 使用\lceil和\rceil表示。⌈x⌉\lceil x \rceil⌈x⌉ ⇒\Rightarrow⇒ $\lceil x \rceil$
向下取整： 使用\lfloor和\rfloor表示。⌊x⌋\lfloor x \rfloor⌊x⌋ ⇒\Rightarrow⇒ $\lfloor x \rfloor$

六、省略号、空白间隔

1、省略号

  • \dots

    使用：$x_1, x_2, \dots, x_n$ ⇒\Rightarrow⇒ x1,x2,…,xnx_1, x_2, \dots, x_nx1​,x2​,…,xn​

  • \cdots

    使用：$1, 2, \cdots, n$ ⇒\Rightarrow⇒ 1,2,⋯ ,n1, 2, \cdots, n1,2,⋯,n

  • \vdots 竖直省略号，一般用于矩阵中。

    使用：$\vdots$ ⇒\Rightarrow⇒ ⋮\vdots⋮

  • \ddots 45o45^o45o 方向省略号，一般用于矩阵中。

    使用：$\ddots$ ⇒\Rightarrow⇒ ⋱\ddots⋱

七、字体

1、使用\it显示意大利体（公式默认字体）：

Abc\it{Abc}Abc ⇒\Rightarrow⇒ $\it{Abc}$

2、使用\mathbb或\Bbb显示黑板粗体（黑板黑体）:

Abc\mathbb{Abc}Abc ⇒\Rightarrow⇒ $\mathbb{Abc}$

3、使用\mathbf或\bf显示黑体：

Abc\mathbf{Abc}Abc ⇒\Rightarrow⇒ $\mathbf{Abc}$

3、使用\mathtt或\tt显示打印机字体：

Abc\mathtt{Abc}Abc ⇒\Rightarrow⇒ $\mathtt{Abc}$
